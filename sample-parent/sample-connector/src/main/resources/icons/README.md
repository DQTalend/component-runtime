Need to be SVG and/or PNG with (Better to provide both with svg2png):
 - PNG must be
   - 32x32
   - named  NAME_iconXX.png
 - SVG
   - 16x16
   - named  NAME_.svg 

For test purpose:
 - family and has PNG and SVG
 - mapper has only SVG
 - output has only PNG
