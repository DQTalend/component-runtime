## TCOMP demo

### Pre-requisite
The backend server should be on `http://localhost:9090`, the frontend dev server has a proxy to this api url

### Build
To build the webapp :

```
yarn
```

or if you don't use yarn

```
npm install
```

### Start

To start the webapp :

```
yarn start
```

or if you don't use yarn

```
npm start
```
